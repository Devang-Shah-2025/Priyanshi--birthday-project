## ğŸ“¦ Deployment to Vercel

   ```bash

   git branch -M main
   ```

2. **Deploy on Vercel:**
   - Go to [vercel.com](https://vercel.com)
   - Click "New Project"
   - Import your GitHub repository
   - Framework Preset: **Next.js** (auto-detected)
   - Click **Deploy**

That's it! Your site will be live in ~1 minute.
